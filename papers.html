How the sync works

Daily GitHub Actions update

A scheduled workflow calls a small Python script that uses the scholarly library to fetch the latest publications from Google Scholar. The script writes everything to assets/data/papers.json and commits the change back to the repository.

What you need to configure
  • Add a SCHOLAR_AUTHOR_ID secret in the repository settings. You'll find your author ID in the Google Scholar profile URL.
  • Optional: customise the schedule or extend the script to enrich papers with PDFs, talk recordings, or BibTeX exports.
