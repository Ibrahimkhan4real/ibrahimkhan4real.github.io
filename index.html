Autonomous Laser Weeding Robot

Project overview

Designed an agricultural robot to autonomously navigate crop fields, detect weeds using computer vision, and remove them using a laser-based actuation system.

Key features

  • Weed–crop discrimination via computer vision using shape, size, and colour features.
  • Integrated a 10 W carbon laser to eliminate weeds with targeted exposure as an alternative to chemical herbicides.
  • Real-time verification of weed removal before resuming navigation.
  • Custom chassis design based on four-bar linkages for stability and mobility on farmland.
  • Autonomous navigation with end-to-end path-planning for crop rows.
  • CNC-based actuation to align the laser precisely above each weed.
